##Installation
Copy the output assembly (DotNetPretty.dll) to 
C:\\Users\\__[username]__\\Documents\\__[Visual Studio Version]__\\Visualizers, 
i.e.: C:\\Users\\__GordonB__\\Documents\\__Visual Studio 2013__\\Visualizers
